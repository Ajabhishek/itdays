Animation

What is animation?

  • Animation is the rapid display of a sequence of images to create an illusion of movement. (WikiPedia)
  • Animation is about change (visual, audio)
  • Cartoons, 3D, etc.

Today's animations

  • Document Object Model (DOM) animation
  • Animation with CSS3 Transitions
  • Use a PNG Sprite Sheet to make animation
  • Particle animation using <canvas>
